Roadmap

  • VarDump 1.0

    - Free release - Basic support just for code types and arrays

  • VarDump 2.0

    - Added functionality for dumping object from code

  • VarDump Plus 1.0

    - Change to paid
    - Re-written module structure
    - Added support for blueprint variables and native structures

  • VarDump Plus 1.1

    - New editor independent in-game overlay
    - Blueprint graph name in witch the dump call was made now shows on the dump name

  • Future

    - New logic to blueprint dumps to take advantage of Unreal reflection system
    - Value histogram
    - VarWatch: automatic update from the dump (without the need to add logic to update)
